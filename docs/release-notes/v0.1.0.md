# BOOTH New Product Discord Notifier Extension v0.1.0

Initial release of the BOOTH new product notifier browser extension.
This browser extension is a port of [moyu1254/booth-new-product-discord-notifier.git](https://github.com/moyu1254/booth-new-product-discord-notifier.git).

## Highlights

- Notify Discord when new BOOTH products appear for configured tags.
- Check on browser startup and on a recurring schedule while the browser is open.
- Support adult-content searches using the active BOOTH browser session.
- Avoid flooding on first setup by marking existing products as seen.
- Show unread counts and recent products in the extension popup.
- Provide separate Chromium and Firefox packages.

## Packages

- `booth-new-product-discord-notifier-extension-chromium-v0.1.0.zip`
- `booth-new-product-discord-notifier-extension-firefox-v0.1.0.zip`
