# BOOTH New Product Discord Notifier Extension v0.1.0

初回リリースです。  
このブラウザ拡張は [moyu1254/booth-new-product-discord-notifier.git](https://github.com/moyu1254/booth-new-product-discord-notifier.git) をブラウザ拡張として移植したものです。

## 主な内容

- 指定した BOOTH タグの新着商品を Discord に通知
- ブラウザ起動時と定期実行に対応
- ブラウザのログイン状態を使った成人向け検索に対応
- 初回設定時の大量通知を避ける既読登録に対応
- 拡張機能ポップアップで未読件数と新着商品を確認可能
- Chromium 版と Firefox 版の両方を提供
