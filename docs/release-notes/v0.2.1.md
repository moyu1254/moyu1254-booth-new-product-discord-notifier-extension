# BOOTH New Product Discord Notifier Extension v0.2.1

セキュリティと保守性の改善リリースです。

## 変更内容

- Discord Webhook URL を同期ストレージではなくローカルストレージに保存するよう変更
- 旧バージョンの同期ストレージ設定をローカルへ移行し、同期側を削除
- Discord Webhook URL の形式検証を追加
- BOOTH 商品解析ロジックを background と offscreen で共通化
- Chromium 向け release package から不要ファイルを除外
- GitHub Actions の書き込み権限を release 用ジョブに限定
